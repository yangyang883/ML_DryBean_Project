# Data Files

This project uses the Dry Bean Dataset multi-class classification task.

Expected files:

- `data/train.csv`
- `data/test.csv`
- Optional: `data/val.csv`

The target column must be named `Class`. All other columns are treated as numeric features.

The current project was initialized from:

- `DryBeanDataset/Dry_Bean_Dataset_Dirty_train.csv`
- `DryBeanDataset/Dry_Bean_Dataset_Dirty_test.csv`
- `DryBeanDataset/Dry_Bean_Dataset_Dirty_val.csv`
